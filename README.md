# Crack the Code

A logic-based number guessing game (similar to Bulls and Cows) available in both C++ (CLI) and a modern Web Interface.

## How to Play
1. The system generates a random **4-digit secret number** (digits 1-9, unique).
2. You make a guess.
3. You get feedback:
   - **Positions (P):** Correct digit in the correct spot.
   - **Numbers (N):** Correct digit but in the wrong spot.
4. Try to find the secret in the fewest attempts!

## Features
* **Dual Mode:** Play Single Player vs AI or Two Player mode with a friend.
* **Responsive Design:** Fully mobile-friendly with a custom on-screen keypad.
* **Visuals:** Cyberpunk aesthetic with animated background and glassmorphism UI.
* **Algorithm:** Custom logic implemented in both C++ and JavaScript.
